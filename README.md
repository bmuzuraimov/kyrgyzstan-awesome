## 🌐 Манифест

> Кыргызстан 2030 — бул санариптик эгемендүүлүгү бар өлкө.

Мы стоим на пороге десятилетия, которое определит цифровое будущее Кыргызстана.
У нас есть разработчики мирового уровня, язык с тысячелетней историей и государство, готовое к трансформации.
Не хватало одного — **единого центра, где всё это собрано воедино**.

Этот список — стратегический актив страны.
Каждый проект здесь — доказательство того, что кыргызстанцы строят, а не только потребляют технологии.

Сегодня инструменты ИИ и агенты-кодировщики — такие как GitHub Copilot, Cursor и Claude — впервые в истории позволяют **одному человеку строить то, что раньше требовало целой команды**. Это меняет правила игры для малых стран с большими амбициями. Студент из региона может сегодня создать инструмент уровня enterprise. Сообщество из десяти человек может построить инфраструктуру для миллиона.

Мы собираем это сообщество здесь.

## Обработка кыргызского языка (NLP)

> Речевые и языковые технологии для кыргызского языка — от распознавания речи до синтеза.

### Модели

- [kyrgyz-ai/Wav2vec-Kyrgyz](https://huggingface.co/kyrgyz-ai/Wav2vec-Kyrgyz) - Модель распознавания речи на основе Wav2Vec для кыргызского языка
- [kyrgyz-ai/AkylAI-STT-small](https://huggingface.co/kyrgyz-ai/AkylAI-STT-small) - Компактная модель распознавания речи (0.2B параметров)
- [kyrgyz-ai/Whisper_STT](https://huggingface.co/kyrgyz-ai/Whisper_STT) - Fine-tuned Whisper для преобразования кыргызской речи в текст
- [kyrgyz-ai/akylai-tts-mini](https://huggingface.co/kyrgyz-ai/akylai-tts-mini) - Мини-модель синтеза речи (TTS) на кыргызском языке
- [kyrgyz-ai/TTS_small](https://huggingface.co/kyrgyz-ai/TTS_small) - Компактная TTS-модель (83M параметров) для кыргызского языка

### Датасеты

- [kyrgyz-ai/asr-fleurs-en-kg-ru-benchmark](https://huggingface.co/datasets/kyrgyz-ai/asr-fleurs-en-kg-ru-benchmark) - ASR-бенчмарк на основе FLEURS для английского, кыргызского и русского языков
- [kyrgyz-ai/result_v0.1_asr-fleurs-en-kg-ru-benchmark](https://huggingface.co/datasets/kyrgyz-ai/result_v0.1_asr-fleurs-en-kg-ru-benchmark) - Результаты бенчмарка v0.1
- [kyrgyz-ai/result_v0.2_asr-fleurs-en-kg-ru-benchmark](https://huggingface.co/datasets/kyrgyz-ai/result_v0.2_asr-fleurs-en-kg-ru-benchmark) - Результаты бенчмарка v0.2
- [kyrgyz-ai/result_v0.3_asr-fleurs-en-kg-ru-benchmark](https://huggingface.co/datasets/kyrgyz-ai/result_v0.3_asr-fleurs-en-kg-ru-benchmark) - Результаты бенчмарка v0.3

## 🚧 Открытые данные и API государства

> Открытые государственные данные — сырьё для следующего поколения кыргызских стартапов.

Полезные порталы (не GitHub):
- [data.gov.kg](https://data.gov.kg) - Официальный портал открытых данных с CKAN API
- [stat.kg](https://stat.kg) - Национальная статистика Кыргызской Республики
- [osoo.kg](https://osoo.kg) - База данных юридических лиц и налоговых данных
- [HDX Kyrgyzstan](https://data.humdata.org/group/kgz) - Субнациональные данные от OCHA

Ищем: библиотеки, обёртки API и парсеры для этих порталов.
